## 1. 고객별 재구매 시기 : 히스토그램
<img src="image/고객별_재구매_시기.png" alt="히스토그램" width="400"/>    
- 고객별 두 번째 구매까지 걸린 날짜(구간 단위)와 고객들의 교차 카운트를 나타냄. </br> 
- 각 막대는 특정 구간에 해당하는 고객 수를 보여주며, 높이가 높을수록 그 구간에 해당하는 고객의 재구매 비율이 높은 것을 의미함</br>
- ex. 그래프에서 100일 구간에 해당하는 막대는 고객들이 두 번째 구매까지 걸린 평균 기간이 100일임을 나타내며, 교차 카운트는 25명</br>
